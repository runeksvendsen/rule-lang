# Build instructions

## Building

1. Install [stack](https://docs.haskellstack.org/en/stable/install_and_upgrade/)
2. Proceed as follows:

```bash
$ git clone https://github.com/runeksvendsen/rule-lang.git
$ cd rule-lang/
$ stack setup
$ stack build
```

## Testing

After building, run `stack test`.
